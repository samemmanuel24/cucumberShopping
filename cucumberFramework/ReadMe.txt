Description:
Automation of http://automationpractice.com using Selenium with cucumber framework
When executing from IDE, reports are generated at 'target/cucumber-reports/index.html'

Prerequisite/Configuration setup required for the project:
JRE/JDK 1.8 or above
Maven 3.1.1
Eclipse IDE

Enhancements:
1.Screenshots can be added to each step.
2.Continous integration of Project can be attain with Jenkins introduction.
 

Execution:
1.Run the Testexec.java using Junit in Eclipse

2.Run from Maven:
Go to terminal or open command prompt
Navigate to Project Path
Run the command 'mvn clean install'
